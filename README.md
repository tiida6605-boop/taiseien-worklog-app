# （株）大成園 作業記録アプリ

HTML / CSS / JavaScript で構成したシングルページPWAです。  
データはブラウザの `localStorage` に保存されます。

## 就業時間の計算

- 標準勤務時間: `7:30〜17:00`
- 標準のカウントしない時間（休憩）: `90分`
- 標準実労働時間: `8.0時間`

作業記録フォームの「勤務時間」では以下を設定できます。

- 開始時刻
- 終了時刻
- カウントしない時間（分）
- 自動計算された労働時間
- 手入力上書き（任意）

「標準時間を入れる」ボタンを押すと、`7:30 / 17:00 / 90分` が自動入力されます。  
手入力上書きを入れた場合は、手入力値を優先して保存します。
