# （株）大成園 作業記録アプリ

> 農作業の「記録」を、  
> これからの「改善」と「分析」につなげるためのスマホ向けアプリ

（株）大成園の現場利用を想定して作成した、農作業記録アプリです。  
毎日の作業内容・作業時間・参加人数・概算人件費などを、スマホで見やすく、押しやすく、扱いやすくまとめることを目的としています。

GitHub Pages で公開しており、ブラウザからすぐ確認できます。

## このアプリで目指していること

農業の現場では、毎日の作業が積み重なって経営になります。  
しかし実際には、

- どの作業にどれだけ時間がかかったか
- 何人で対応したか
- 人件費がどの程度かかったか
- どの日にどんな作業が集中したか

が感覚ベースになりやすく、後から振り返りにくいことがあります。

このアプリは、そうした日々の作業を**記録・見える化**し、  
将来的には**集計・改善・AI分析**へつなげるための土台として作成しています。

---

## 主な機能

### 1. ホーム画面で今日の状況を確認
- 今日の作業件数
- 今日の総作業時間
- 今日の総延べ人時
- 今日の総概算人件費
- 今日のチーム編成件数

その日の状況を、まず一目で把握できるようにしています。

### 2. 作業を記録する
現場で行った作業を入力・記録するための機能です。  
「まず記録する」を起点に使う設計を意識しています。

### 3. 履歴を見る
過去の記録を確認し、いつ・どんな作業を行ったか振り返るための画面です。

### 4. 今日のメンバーを見る
当日の参加者や作業メンバーの確認をしやすくするための機能です。

### 5. まとめを見る
記録した内容を一覧や集計の形で確認するための画面です。

### 6. スマホ向けの操作性
ボタンを大きめに配置し、現場で迷いにくい構成を目指しています。  
家族や従業員でも直感的に扱えることを重視しています。

### 7. PWA対応
- `manifest.json`
- `sw.js`
- `offline.html`

を含んでおり、アプリ風の利用やオフライン対応を意識した構成になっています。

## 想定する利用シーン

- りんご園の日々の作業記録
- 剪定、摘果、防除、草刈り、収穫などの工程管理
- 作業別の時間把握
- 人員配置の見直し
- 人件費の概算把握
- データを蓄積して、後から分析・改善につなげる運用

## このアプリの特徴

このアプリは、単なるメモではなく、  
**「現場で使える記録」→「あとから集計できる記録」**  
にしていくことを重視しています。

将来的には、次のような発展も視野に入れています。

- 園地別の作業分析
- 品種別の作業時間比較
- 作業効率の見直し
- 年間の人手配分の把握
- AIによる傾向分析
- 補助事業や経営資料づくりへの活用
