# (株)大成園 作業記録アプリ

HTML / CSS / JavaScript のシングルページ PWA です。  
データは `localStorage` に保存されます。

## 就業時間の計算

- 標準勤務は `7:30〜17:00`
- 標準勤務は昼休み・休憩時間として `120分` を控除
- 標準勤務の給与計算に使う実労働時間は `7.5時間`
- 午前のみ・午後のみは `15分` 控除
- 変則勤務は開始時刻・終了時刻・控除時間を入力して計算
- 勤務区分に「手入力」を選ぶと、実労働時間を直接入力可能
- 作業者ごとに勤務区分・開始/終了・控除時間・手入力時間を調整可能

計算式:

- 拘束時間 = 終了時刻 - 開始時刻
- 実労働時間 = 拘束時間 - 控除時間
- 給与 = 実労働時間 × 時給

## 簡易給与明細

- 月締め / 半月締め（前半・後半） / 任意期間に対応
- 日付ごとの明細を表示
  - 日付、曜日、作業内容
  - 開始時刻 / 終了時刻 / 控除時間
  - 実労働時間 / 時給 / 日別賃金
- 同日の複数記録は日単位で合算
- 最後に期間合計（出勤日数、総実労働時間、総控除時間、総支給額）を表示
- 印刷 / PDF保存 / CSV出力に対応
- 簡易給与明細は控除後の実労働時間で計算
